# Glossary

- AO: Audit Object
- OV: Observable Variable
- EA: Evidence Artifact (instance)
- E1-E4: Evidence tiers (cryptographic, measurement, configuration, process/provenance)
- TCB: Trusted Computing Base
- TEE: Trusted Execution Environment
- SE: Secure Element
- AVB: Android Verified Boot
- RATS: IETF Remote Attestation Procedures
- SBOM: Software Bill of Materials
- SLSA: Supply-chain Levels for Software Artifacts
